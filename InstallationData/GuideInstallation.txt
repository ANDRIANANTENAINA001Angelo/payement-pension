########## GUIDE INSTALLATION, CONFIGURATION ET LANCEMENT APPLICATION ##########

1- Extraire php8.zip
2- Ajouter chemin php8 dans variable environement système windows
3- Vérification version php après Installation avec php --version (résultat: PHP 8.2.5 (cli) (built: Apr 12 2023 08:42:33) (NTS Visual C++ 2019 x64) Copyright (c) The PHP GroupZend Engine v4.2.5, Copyright (c) Zend Technologies)
4- Installation de node 
5- Vérification version node après Installation avec node --version (résultat : v18.17.0)
6- Installation composer 
7- Vérification version composer après Installation avec composer --version (résultat: Composer version 2.5.8 2023-06-09 17:13:21)
8- Installation base de donnée Mysql (xampp ou wampp)
9- lancer base de donnée Mysql


10- ouvrir terminal dans le dossier .\laravel-app 
11- lancer cmd : composer install (mila connexion)
12- lancer cmd : npm install (mila connexion)
13- lancer cmd : php artisan migrate (oui pour créé base de donnée et les tables de l'application )
14- ouvrir autre terminal et lancer cmd : npm run dev 


15- ouvrir navigateur, taper http://localhost:8000/register (pour créé compte )
16- ouvrir navigateur, taper http://localhost:8000/admin/login (pour se connecter )

17- importer fichier Excel pensionnaire et utiliser application
